# 🌿 Organic India — Agricultural Research Platform

An interactive, data-rich web platform exploring the history, transition, and future of organic agriculture in India. This project analyzes whether the shift to organic farming is a sustainable revolution or an unsustainable concept for a nation of 1.4 billion people.

## 🚀 Features

* **Interactive Data Visualization:** Dynamic charts (Bar, Doughnut, Line) built with `Chart.js` to visualize export economics, state-wise adoption, and yield transition curves.
* **Domain-Specific AI Chatbot:** A custom-built floating AI assistant that answers user queries specifically based on the site's research data, history, and developer information.
* **Deep-Dive Slide Panels:** Clickable UI cards that trigger smooth, slide-in panels containing heavily researched data, historical context, and high-quality imagery.
* **Direct WhatsApp Integration:** A "Propose Your Solution" form that securely formats user ideas and sends them directly to the developer's WhatsApp for instant collaboration.
* **Modern Glassmorphism UI:** Features a highly responsive design with transparent frosted-glass elements, custom scrollbars, and hover animations.
* **Modular PHP Architecture:** Designed using PHP to allow for scalable templating and future backend integrations.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** PHP 8.x
* **Libraries:** [Chart.js](https://www.chartjs.org/) (for data visualization)
* **Design/UI:** Custom CSS (Glassmorphism, Flexbox, CSS Grid)
